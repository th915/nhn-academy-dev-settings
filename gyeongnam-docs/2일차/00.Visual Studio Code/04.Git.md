- GUI

![](./images/git-on-vsc/03.add-file.png)

### 3. Commit changes

#### Try Commit

- GUI

![](./images/git-on-vsc/04.commit-changes.png)

- Command

```

```

#### Set Git User info

- GUI

![](./images/git-on-vsc/05.set-git-user.png)

- Command

```
git config --global user.email "you@example.com"
git config --global user.name "Your Name"
```
